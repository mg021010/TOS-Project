스케줄러 설정 및 제어 명령어 인터페이스

💡 구현 내용

  • 스케줄러 동작을 설정하고 틱 단위 시뮬레이션을 수행하기 위한 인터페이스 확장.
  • set_sched <fcfs|rr> [quantum]: 스케줄링 알고리즘 변경 및 RR 퀀텀 시간 동적 지정.

  • stats & reset: 실행 통계(대기시간, 컨텍스트 스위칭 수) 확인 및 시뮬레이션 환경 초기화.

Step 3

FCFS (First-Come, First-Served) 스케줄링

💡 구현 내용

  • 준비 큐(Ready Queue)에 먼저 삽입된 프로세스에 CPU를 우선 할당하는 비선점형 스케줄러 구현.
  • 중간에 CPU를 빼앗기지 않고 프로세스의 남은 실행 시간(Remaining Time)이 0이 될 때까지 연속 실행.
  • CPU를 양보할 때 준비 큐의 맨 앞에서 다음 프로세스를 디스패치하며 문맥 교환 발생.
  • 실행 중이지 않고 Ready Queue에 대기 중인 모든 프로세스의 대기 시간(Waiting Time)을 매 틱마다 1씩 증가시켜 통계 신뢰도 확보.
create P1 5 create P2 3 set_sched fcfs run stats exit

Step 4

I/O 대기 및 대기 큐(Waiting Queue) 구현

💡 구현 내용

  • 프로세스가 실행되는 중 I/O 요청을 보내고 대기 상태에 머무는 OS 동작 흐름 시뮬레이션.
  • create <name> <time> io <at> <wait>: at 시점에 wait 초 동안 I/O를 사용하는 프로세스 생성.
  • 프로세스가 실행 중 누적 실행시간이 io_at에 도달하면 즉시 PROCESS_WAITING 상태로 전이되어 대기 큐(Waiting Queue)로 이동.
  • 매 틱마다 대기 큐 안의 프로세스들의 남은 I/O 시간(io_remaining)을 1씩 감소시키며, 0에 도달하면 PROCESS_READY로 복귀시켜 준비 큐에 재진입.
create P1 5 io 2 3 set_sched fcfs run stats exit
📸
I/O 대기 실행 결과 캡처 영역

위 명령어를 실행하여 P1이 Time 2에 Waiting Queue로 이동하고, I/O Completed 후 복귀하여 완료하는 로그를 캡처하여 images/step4_io.png로 저장하세요.

Step 5

Round-Robin (RR) 선점형 스케줄링

💡 구현 내용

  • 모든 프로세스가 일정한 타임 슬라이스(Quantum) 동안 동등하게 CPU를 공유하는 선점형 스케줄링.
  • 현재 프로세스의 연속 실행 시간이 설정된 타임 퀀텀(기본 3)에 도달하면 실행을 강제 중단(Preemption)시킴.
  • 해당 프로세스의 상태를 PROCESS_READY로 변경하고 준비 큐의 맨 뒤에 다시 삽입.
  • 타임 슬라이스 소진, 프로세스 완료, 혹은 I/O 블로킹 발생 시에 즉각 문맥 교환을 수행하여 CPU 분할 사용이 최적으로 수행됨을 검증.
create P1 5 create P2 3 set_sched rr 2 run stats exit
📸
Round-Robin 실행 결과 캡처 영역

위 명령을 실행하여 P1이 2틱 후 선점(Preempt)되고 P2로 교체되는 라운드 로빈 교차 진행 화면을 캡처하여 images/step5_rr.png로 저장하세요.

Step 6

가상 메모리 페이징 및 Page Fault 시뮬레이션

💡 구현 내용

  • 물리 RAM 1KB (프레임 크기 128B, 총 8개 프레임) 및 가상 페이지 테이블(최대 16개 엔트리) 구조 시뮬레이션.
  • memsim <scenario_file> 명령어로 시나리오 파일을 순차 해석.
  • ALLOC <page>: 비어 있는 물리 메모리 프레임을 검색하여 페이지 테이블에 유효 비트(is_valid=1)와 함께 순차 매핑.
  • ACCESS <virtual_addr>: 주소 변환(Address Translation) 수행.
    • 가상 페이지 번호(VA / Page Size)가 유효(Valid)할 시, 매핑된 프레임 번호로 변환하여 물리 주소(PA) 계산.
    • 매핑되지 않은 가상 페이지 접근 시 Page Fault 예외 발생.
cat scenario.txt memsim scenario.txt exit

메모리 시뮬레이션 결과 캡처 영역
